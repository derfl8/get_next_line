**Description :**

Mandatory part :

This project is a set of functions. His reach is to open a text file, and split it line by line. Return all lines and split them one by one with some different buffer size (a number of characters per call).

Bonus part :

You need to allow from 1 to 1024 different text files.


**Instructions :**

For compiling you need to add after flag : -D BUFFER_SIZE(you can add ="number" to custom the buffer size). You also need to add a main. Without main you can't run this project.
